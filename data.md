# Datasets and Notebooks

This section provides access to the reference datasets and training notebooks required for the DUA mapping pipeline.

## Datasets
* **Sentinel-2 Stacks:** Multi-spectral imagery for urban regions.
* **Reference Labels:** Ground-truth vector data for training and validation.
* **Footprints:** Standardized building footprint data used for spatial constraints.

## Notebooks
Access the interactive environment to run the pipeline or explore model outputs:

* **[Launch Binder](https://mybinder.org/v2/gh/YOUR_USER/YOUR_REPO/main):** Launch an interactive environment pre-configured with the project dependencies.
* **[JupyterLite](link-to-your-instance):** Run the notebooks directly in your browser without any local setup.
